Authoring Plugins¶

Here we try to provide any additional documentation one will need to create their own custom plugins to use with BuildStream.

Core Framework¶

The core public APIs are of interest to anyone who wishes to implement custom Element or Source plugins.

  • Plugin - Base Class for all plugins
  • Source - Base Source Class
  • Element - Base Element Class
  • BuildElement - Build Element Class
  • ScriptElement - Script Element Class
  • Sandbox - Build Sandbox
  • Utilities - Utilities for Plugins
